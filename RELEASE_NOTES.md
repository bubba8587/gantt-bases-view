## What's New

- **Compact rows** — a new toolbar toggle packs tasks that don't overlap in time onto shared rows, so an FS chain reads as one continuous line instead of a staircase. Within each group, bars are packed left-to-right into the fewest lanes that avoid collisions (milestone diamonds reserve their painted overhang). The sidebar shows one summary cell per group with a task count — labels live on the bars themselves — and dependency arrows, hover chains, drag, locks, and editing all work unchanged. Tasks without dates aren't drawable in compact mode; the sidebar cell notes how many are hidden.
- **Roomier edit popup** — widened to fit the Start / End / Days row comfortably; the day count takes a narrow fixed column so the date inputs get the space.

No changes to frontmatter format, settings, or vault data.
